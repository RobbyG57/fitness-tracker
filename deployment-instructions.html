Manual GitHub Pages Deployment Instructions

Your Fitness Tracker app build is ready for GitHub Pages! Follow these steps to deploy it:

Steps to deploy:

  1. Go to your GitHub repository: https://github.com/RobbyG57/fitness-tracker
  2. Click on "Settings" (top navigation bar)
  3. Scroll down to "GitHub Pages" section under "Code and automation" in the sidebar
  4. Under "Source", select "Deploy from a branch"
  5. Create a new branch called gh-pages
  6. In the dropdown, select this new gh-pages branch and click "Save"
  7. Upload all files from the gh-pages-temp folder in your project to this branch:
    • You can do this by clicking "Add file" > "Upload files" on the gh-pages branch page
    • Drag and drop all files from the gh-pages-temp folder
    • Click "Commit changes"
  8. Wait a few minutes for GitHub to build your site
  9. Visit https://RobbyG57.github.io/fitness-tracker to see your deployed app!

Special Note: Make sure both index.html and 404.html files are in the root of your gh-pages branch. This is critical for proper SPA routing!

Note: For future updates, you'll need to rebuild your app with npm run build and repeat this process.
